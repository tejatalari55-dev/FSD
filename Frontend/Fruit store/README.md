# 🍎 FruityDelight - The Fun Fruit Store! 🌈

Welcome to **FruityDelight**, a vibrant and playful web-based fruit store designed to bring joy to your shopping experience! This project features a colorful, interactive UI with glassmorphism effects and delightful animations.

## ✨ Features

- 🛒 **Interactive Cart:** Add your favorite fruits to your bag and watch them fly into the cart!
- 🌓 **Day/Night Mode:** Toggle between a bright, sunny theme and a cool, dark theme.
- 🔍 **Search & Filter:** Easily find fruits by name or browse through categories like Tropical, Citrus, and Berries.
- 📱 **Responsive Design:** Optimized for both desktop and mobile devices.
- 🎈 **Playful Animations:** Enjoy floating background fruits and bouncy interactions.
- 🍎 **Fruit Facts:** Learn fun facts about each fruit in the detailed view modal!

## 🚀 How to Run

No installation required! Just follow these simple steps:

1. Download or clone the project files.
2. Open `index.html` in any modern web browser.
3. Start shopping! 🛍️

## 📂 Project Structure

- `index.html`: The main structure of the store.
- `styles.css`: The styling that makes everything look colorful and "glassy".
- `app.js`: The logic behind the cart, filtering, and animations.
