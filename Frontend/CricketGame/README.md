# Simple Cricket Game
A simple web-based game based on the logic of Rock-Paper-Scissors, themed around Cricket.
## How to Play
1. Open the game in your browser.
2. Click on one of the three options: **Bat**, **Ball**, or **Stump**.
3. The computer will randomly make its own choice.
4. The result (Win, Loss, or Tie) will be displayed along with your current score.
5. Use the **Reset** button to clear your score and start over.

## How to Run
- **Windows:** Double-click the `run.bat` file to launch the game.
- **Other Platforms:** Open `cricket.html` directly in any modern web browser.

## Project Structure
- `cricket.html`: The main game logic and structure.
- `cricket.css`: Basic styling for the game interface.
- `images/`: Contains the icons for Bat, Ball, and Stumps.
- `run.bat`: A shortcut to open the game on Windows.
